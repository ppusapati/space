## 1. Executive Summary

### 1.1 Purpose

This document provides the detailed technical specification for the Telemetry Pipeline, the core subsystem responsible for:

- Receiving raw telemetry frames from ground station equipment
- Validating, parsing, and extracting parameters
- Calibrating raw values to engineering units
- Checking limits and generating alerts
- Storing time-series data efficiently
- Streaming real-time data to clients
- Supporting historical queries and analysis

### 1.2 Design Goals

| Goal | Target | Rationale |
|------|--------|-----------|
| **Low Latency** | < 500ms end-to-end | Real-time monitoring |
| **High Throughput** | 10,000 params/sec | Multiple satellites |
| **Scalability** | 100+ satellites | Fleet operations |
| **Reliability** | 99.9% uptime | Mission critical |
| **Data Retention** | 5+ years | Long-term analysis |
| **Query Performance** | < 1s for 24h data | Operational efficiency |

### 1.3 Key Technologies

| Component | Technology | Rationale |
|-----------|------------|-----------|
| Message Broker | NATS JetStream | Low latency, persistence |
| Time-Series DB | TimescaleDB | PostgreSQL compatible, compression |
| Cache | Redis | Real-time state, pub/sub |
| Object Storage | S3/MinIO | Cold archive, exports |
| Streaming | WebSocket | Browser compatibility |
| Processing | Go | Performance, concurrency |

### 3.2 Data Transformation Flow

```
Stage 1: Frame Reception
────────────────────────
Input:  Raw bytes from modem/TNC
Output: RawFrame { timestamp, station_id, bytes[] }

Stage 2: Frame Validation
─────────────────────────
Input:  RawFrame
Output: ValidatedFrame { timestamp, satellite_id, frame_type, payload[] }
Actions:
  - Sync word detection
  - CRC/checksum verification
  - Frame type identification
  - Satellite ID extraction

Stage 3: Frame Decommutation
────────────────────────────
Input:  ValidatedFrame
Output: DecommutatedFrame { parameters[] }
Actions:
  - Apply packet definition (ICD)
  - Extract raw parameter values
  - Handle variable-length fields
  - Process sub-commutated data

Stage 4: Parameter Calibration
──────────────────────────────
Input:  RawParameter { name, raw_value, type }
Output: CalibratedParameter { name, eng_value, unit, quality }
Actions:
  - Apply calibration curve
  - Convert to engineering units
  - Calculate derived parameters
  - Set quality flags

Stage 5: Limit Checking
───────────────────────
Input:  CalibratedParameter
Output: ParameterWithStatus { ..., limit_status, alarm_state }
Actions:
  - Compare against thresholds
  - Apply persistence filtering
  - Check rate-of-change
  - Generate/clear alarms

Stage 6: Storage & Distribution
───────────────────────────────
Input:  ParameterWithStatus
Output: Stored record, WebSocket message
Actions:
  - Write to TimescaleDB
  - Update Redis current value
  - Publish to subscribers
  - Archive if needed
```

### 4.3 Frame Receiver Implementation

```go
// frame_receiver.go

package telemetry

import (
    "context"
    "time"
    "github.com/nats-io/nats.go"
)

// RawFrame represents a frame as received from ground station
type RawFrame struct {
    Timestamp   time.Time `json:"timestamp"`
    StationID   string    `json:"station_id"`
    PassID      string    `json:"pass_id,omitempty"`
    Protocol    string    `json:"protocol"`  // "CCSDS_TM", "AX25", "CUSTOM"
    RawBytes    []byte    `json:"raw_bytes"`
    RSSI        float64   `json:"rssi,omitempty"`        // Signal strength
    FreqOffset  float64   `json:"freq_offset,omitempty"` // Hz
}

// FrameReceiver handles incoming telemetry from a ground station
type FrameReceiver struct {
    stationID  string
    protocol   FrameProtocol
    natsConn   *nats.Conn
    jetStream  nats.JetStreamContext
    metrics    *ReceiverMetrics
    buffer     *CircularBuffer
}

// FrameProtocol defines the interface for protocol handlers
type FrameProtocol interface {
    // ReadFrame reads the next complete frame
    ReadFrame(ctx context.Context) ([]byte, error)
    // GetSyncWord returns the sync pattern for this protocol
    GetSyncWord() []byte
    // ValidateFrame performs protocol-specific validation
    ValidateFrame(data []byte) error
}

// NewFrameReceiver creates a new frame receiver for a ground station
func NewFrameReceiver(cfg ReceiverConfig) (*FrameReceiver, error) {
    // Connect to NATS
    nc, err := nats.Connect(cfg.NatsURL)
    if err != nil {
        return nil, fmt.Errorf("NATS connection failed: %w", err)
    }
    
    js, err := nc.JetStream()
    if err != nil {
        return nil, fmt.Errorf("JetStream init failed: %w", err)
    }
    
    // Create protocol handler
    var protocol FrameProtocol
    switch cfg.Protocol {
    case "CCSDS_TM":
        protocol = NewCCSDSTMProtocol(cfg.FrameLength, cfg.SyncWord)
    case "AX25":
        protocol = NewAX25Protocol()
    default:
        protocol = NewCustomProtocol(cfg.FrameDefinition)
    }
    
    return &FrameReceiver{
        stationID: cfg.StationID,
        protocol:  protocol,
        natsConn:  nc,
        jetStream: js,
        metrics:   NewReceiverMetrics(cfg.StationID),
        buffer:    NewCircularBuffer(cfg.BufferSize),
    }, nil
}

// Run starts the frame receiver loop
func (fr *FrameReceiver) Run(ctx context.Context) error {
    subject := fmt.Sprintf("telemetry.frames.%s", fr.stationID)
    
    for {
        select {
        case <-ctx.Done():
            return ctx.Err()
        default:
            // Read frame from protocol handler
            frameBytes, err := fr.protocol.ReadFrame(ctx)
            if err != nil {
                fr.metrics.FrameErrors.Inc()
                continue
            }
            
            // Create raw frame message
            frame := RawFrame{
                Timestamp:  time.Now().UTC(),
                StationID:  fr.stationID,
                Protocol:   fr.protocol.Name(),
                RawBytes:   frameBytes,
            }
            
            // Publish to NATS
            data, _ := json.Marshal(frame)
            _, err = fr.jetStream.Publish(subject, data)
            if err != nil {
                fr.metrics.PublishErrors.Inc()
                // Buffer for retry
                fr.buffer.Push(data)
                continue
            }
            
            fr.metrics.FramesReceived.Inc()
            fr.metrics.BytesReceived.Add(float64(len(frameBytes)))
        }
    }
}
```

### 4.4 CRC Validation

```go
// crc.go

package telemetry

// CRC-16-CCITT polynomial
const crcPolynomial uint16 = 0x1021

// CalculateCRC16 computes CRC-16-CCITT
func CalculateCRC16(data []byte) uint16 {
    crc := uint16(0xFFFF)
    
    for _, b := range data {
        crc ^= uint16(b) << 8
        for i := 0; i < 8; i++ {
            if crc&0x8000 != 0 {
                crc = (crc << 1) ^ crcPolynomial
            } else {
                crc <<= 1
            }
        }
    }
    
    return crc
}

// ValidateFrameCRC checks if frame CRC is valid
func ValidateFrameCRC(frame []byte) bool {
    if len(frame) < 2 {
        return false
    }
    
    // CRC is last 2 bytes
    dataLen := len(frame) - 2
    data := frame[:dataLen]
    receivedCRC := uint16(frame[dataLen])<<8 | uint16(frame[dataLen+1])
    
    calculatedCRC := CalculateCRC16(data)
    
    return receivedCRC == calculatedCRC
}
```

### 5.3 Calibration Types

```go
// calibration.go

package telemetry

import (
    "math"
    "sort"
)

// Calibrator converts raw values to engineering units
type Calibrator interface {
    Calibrate(rawValue float64) float64
}

// PolynomialCalibrator: y = c0 + c1*x + c2*x^2 + ...
type PolynomialCalibrator struct {
    Coefficients []float64
}

func (p *PolynomialCalibrator) Calibrate(raw float64) float64 {
    result := 0.0
    for i, coef := range p.Coefficients {
        result += coef * math.Pow(raw, float64(i))
    }
    return result
}

// PointPairCalibrator: linear interpolation between points
type PointPairCalibrator struct {
    Points [][2]float64 // [[raw1, eng1], [raw2, eng2], ...]
}

func (pp *PointPairCalibrator) Calibrate(raw float64) float64 {
    // Sort points by raw value
    sort.Slice(pp.Points, func(i, j int) bool {
        return pp.Points[i][0] < pp.Points[j][0]
    })
    
    // Find bracketing points
    for i := 0; i < len(pp.Points)-1; i++ {
        if raw >= pp.Points[i][0] && raw <= pp.Points[i+1][0] {
            // Linear interpolation
            x0, y0 := pp.Points[i][0], pp.Points[i][1]
            x1, y1 := pp.Points[i+1][0], pp.Points[i+1][1]
            return y0 + (raw-x0)*(y1-y0)/(x1-x0)
        }
    }
    
    // Extrapolate if outside range
    if raw < pp.Points[0][0] {
        return pp.Points[0][1]
    }
    return pp.Points[len(pp.Points)-1][1]
}

// LookupTableCalibrator: discrete value mapping
type LookupTableCalibrator struct {
    Table map[int64]float64
    Default float64
}

func (lt *LookupTableCalibrator) Calibrate(raw float64) float64 {
    if val, ok := lt.Table[int64(raw)]; ok {
        return val
    }
    return lt.Default
}

// EnumCalibrator: maps raw to string states
type EnumCalibrator struct {
    Mapping map[int64]string
    Default string
}

func (e *EnumCalibrator) CalibrateToString(raw float64) string {
    if val, ok := e.Mapping[int64(raw)]; ok {
        return val
    }
    return e.Default
}
```

```go
// Quality flags for parameter values
type Quality struct {
    Valid       bool   `json:"valid"`
    Stale       bool   `json:"stale"`
    OutOfLimits bool   `json:"out_of_limits"`
    Source      string `json:"source"` // "realtime", "playback", "simulated"
}

// ParameterProcessor handles decommutation and calibration
type ParameterProcessor struct {
    definitions map[string]*PacketDefinition
    calibrators map[string]Calibrator
    natsConn    *nats.Conn
    jetStream   nats.JetStreamContext
    metrics     *ProcessorMetrics
}

// ProcessFrame decommutates and calibrates a validated frame
func (pp *ParameterProcessor) ProcessFrame(ctx context.Context, frame *ValidatedFrame) ([]CalibratedParameter, error) {
    // Get packet definition
    def, ok := pp.definitions[frame.PacketID]
    if !ok {
        return nil, fmt.Errorf("unknown packet ID: %s", frame.PacketID)
    }
    
    params := make([]CalibratedParameter, 0, len(def.Parameters))
    
    for _, paramDef := range def.Parameters {
        // Extract raw value
        rawValue, err := pp.extractRawValue(frame.Payload, paramDef)
        if err != nil {
            pp.metrics.ExtractionErrors.Inc()
            continue
        }
        
        // Calibrate
        calibrator := pp.calibrators[paramDef.Name]
        engValue := calibrator.Calibrate(rawValue)
        
        param := CalibratedParameter{
            Timestamp:   frame.Timestamp,
            SatelliteID: frame.SatelliteID,
            PassID:      frame.PassID,
            Name:        paramDef.Name,
            Mnemonic:    paramDef.Mnemonic,
            RawValue:    rawValue,
            EngValue:    engValue,
            Unit:        paramDef.Unit,
            Quality: Quality{
                Valid:  true,
                Source: "realtime",
            },
            Subsystem: paramDef.Subsystem,
        }
        
        params = append(params, param)
    }
    
    // Process derived parameters
    derivedParams := pp.calculateDerivedParameters(params, def.DerivedParameters)
    params = append(params, derivedParams...)
    
    // Publish to NATS
    for _, param := range params {
        subject := fmt.Sprintf("telemetry.params.%s.%s", 
            param.SatelliteID, param.Subsystem)
        data, _ := json.Marshal(param)
        pp.jetStream.Publish(subject, data)
    }
    
    pp.metrics.ParametersProcessed.Add(float64(len(params)))
    
    return params, nil
}

// extractRawValue extracts a raw value from frame payload
func (pp *ParameterProcessor) extractRawValue(payload []byte, def *ParameterDefinition) (float64, error) {
    // Handle bit-level extraction
    startByte := def.ByteOffset
    endByte := startByte + (def.BitOffset+def.BitLength+7)/8
    
    if endByte > len(payload) {
        return 0, fmt.Errorf("parameter extends beyond payload")
    }
    
    // Extract bytes
    bytes := payload[startByte:endByte]
    
    // Convert based on byte order
    var rawInt uint64
    if def.ByteOrder == "big_endian" {
        rawInt = binary.BigEndian.Uint64(padBytes(bytes, 8))
    } else {
        rawInt = binary.LittleEndian.Uint64(padBytes(bytes, 8))
    }
    
    // Apply bit offset and mask
    rawInt >>= uint(def.BitOffset)
    mask := uint64((1 << def.BitLength) - 1)
    rawInt &= mask
    
    // Handle signed values
    if def.IsSigned && rawInt&(1<<(def.BitLength-1)) != 0 {
        rawInt |= ^mask // Sign extend
    }
    
    return float64(rawInt), nil
}
```

```go
// limit_checker.go

package telemetry

import (
    "context"
    "sync"
    "time"
)

// LimitStatus represents the limit state of a parameter
type LimitStatus string

const (
    LimitNominal      LimitStatus = "NOMINAL"
    LimitWarningLow   LimitStatus = "WARNING_LOW"
    LimitWarningHigh  LimitStatus = "WARNING_HIGH"
    LimitCriticalLow  LimitStatus = "CRITICAL_LOW"
    LimitCriticalHigh LimitStatus = "CRITICAL_HIGH"
)

// Alarm represents an active limit violation
type Alarm struct {
    ID            string      `json:"id"`
    SatelliteID   string      `json:"satellite_id"`
    Parameter     string      `json:"parameter"`
    Mnemonic      string      `json:"mnemonic"`
    Status        LimitStatus `json:"status"`
    Value         float64     `json:"value"`
    Threshold     float64     `json:"threshold"`
    Unit          string      `json:"unit"`
    Message       string      `json:"message"`
    Severity      string      `json:"severity"`
    RaisedAt      time.Time   `json:"raised_at"`
    AcknowledgedAt *time.Time `json:"acknowledged_at,omitempty"`
    AcknowledgedBy string     `json:"acknowledged_by,omitempty"`
    ClearedAt     *time.Time  `json:"cleared_at,omitempty"`
}

// LimitChecker monitors parameters against defined limits
type LimitChecker struct {
    definitions   map[string]*LimitDefinition
    persistence   map[string]*PersistenceTracker
    rateTrackers  map[string]*RateOfChangeTracker
    activeAlarms  map[string]*Alarm
    mu            sync.RWMutex
    natsConn      *nats.Conn
    jetStream     nats.JetStreamContext
    redis         *redis.Client
    metrics       *LimitMetrics
}

// PersistenceTracker tracks consecutive limit violations
type PersistenceTracker struct {
    Status       LimitStatus
    Count        int
    RequiredCount int
    LastValue    float64
    LastTime     time.Time
}

// RateOfChangeTracker monitors parameter rate of change
type RateOfChangeTracker struct {
    Values    []float64
    Times     []time.Time
    MaxLength int
}

// CheckParameter evaluates a parameter against its limits
func (lc *LimitChecker) CheckParameter(ctx context.Context, param *CalibratedParameter) (*LimitStatus, *Alarm, error) {
    lc.mu.Lock()
    defer lc.mu.Unlock()
    
    def, ok := lc.definitions[param.Name]
    if !ok || !def.Enabled {
        return nil, nil, nil
    }
    
    // Determine current limit status
    status := lc.evaluateThresholds(param.EngValue, def)
    
    // Check persistence
    tracker := lc.getOrCreateTracker(param.SatelliteID, param.Name)
    confirmed := lc.checkPersistence(tracker, status, param.EngValue, def)
    
    // Check rate of change
    if def.RateOfChange != nil && def.RateOfChange.Enabled {
        rateAlarm := lc.checkRateOfChange(param, def.RateOfChange)
        if rateAlarm != nil {
            lc.raiseAlarm(ctx, rateAlarm)
        }
    }
    
    // Handle alarm state transitions
    var alarm *Alarm
    if confirmed {
        alarmKey := fmt.Sprintf("%s.%s", param.SatelliteID, param.Name)
        
        if status != LimitNominal {
            // Raise or update alarm
            existing := lc.activeAlarms[alarmKey]
            if existing == nil || existing.Status != status {
                alarm = lc.createAlarm(param, status, def)
                lc.activeAlarms[alarmKey] = alarm
                lc.raiseAlarm(ctx, alarm)
            }
        } else {
            // Clear alarm if returning to nominal
            if existing := lc.activeAlarms[alarmKey]; existing != nil {
                now := time.Now().UTC()
                existing.ClearedAt = &now
                lc.clearAlarm(ctx, existing)
                delete(lc.activeAlarms, alarmKey)
            }
        }
    }
    
    // Update parameter with limit status
    param.Quality.OutOfLimits = status != LimitNominal
    
    return &status, alarm, nil
}

// evaluateThresholds determines limit status from value
func (lc *LimitChecker) evaluateThresholds(value float64, def *LimitDefinition) LimitStatus {
    if def.CriticalLow != nil && value < *def.CriticalLow {
        return LimitCriticalLow
    }
    if def.WarningLow != nil && value < *def.WarningLow {
        return LimitWarningLow
    }
    if def.CriticalHigh != nil && value > *def.CriticalHigh {
        return LimitCriticalHigh
    }
    if def.WarningHigh != nil && value > *def.WarningHigh {
        return LimitWarningHigh
    }
    return LimitNominal
}

// checkPersistence verifies violation is sustained
func (lc *LimitChecker) checkPersistence(tracker *PersistenceTracker, status LimitStatus, value float64, def *LimitDefinition) bool {
    if tracker.Status != status {
        // Status changed, reset counter
        tracker.Status = status
        tracker.Count = 1
    } else {
        tracker.Count++
    }
    
    tracker.LastValue = value
    tracker.LastTime = time.Now().UTC()
    
    // Get required persistence for this status
    required := def.GetPersistence(status)
    
    return tracker.Count >= required
}

// checkRateOfChange detects rapid parameter changes
func (lc *LimitChecker) checkRateOfChange(param *CalibratedParameter, def *RateOfChangeDefinition) *Alarm {
    key := fmt.Sprintf("%s.%s", param.SatelliteID, param.Name)
    tracker := lc.rateTrackers[key]
    if tracker == nil {
        tracker = &RateOfChangeTracker{MaxLength: 10}
        lc.rateTrackers[key] = tracker
    }
    
    // Add new sample
    tracker.Values = append(tracker.Values, param.EngValue)
    tracker.Times = append(tracker.Times, param.Timestamp)
    
    // Trim to max length
    if len(tracker.Values) > tracker.MaxLength {
        tracker.Values = tracker.Values[1:]
        tracker.Times = tracker.Times[1:]
    }
    
    // Need at least 2 samples
    if len(tracker.Values) < 2 {
        return nil
    }
    
    // Calculate rate of change
    n := len(tracker.Values)
    deltaValue := tracker.Values[n-1] - tracker.Values[n-2]
    deltaTime := tracker.Times[n-1].Sub(tracker.Times[n-2]).Seconds()
    
    if deltaTime == 0 {
        return nil
    }
    
    rate := math.Abs(deltaValue / deltaTime)
    
    if rate > def.MaxChangePerSecond {
        return &Alarm{
            ID:          uuid.New().String(),
            SatelliteID: param.SatelliteID,
            Parameter:   param.Name,
            Mnemonic:    param.Mnemonic,
            Status:      LimitWarningHigh, // Rate alarms are typically warnings
            Value:       rate,
            Threshold:   def.MaxChangePerSecond,
            Unit:        param.Unit + "/s",
            Message:     def.Message,
            Severity:    def.Severity,
            RaisedAt:    time.Now().UTC(),
        }
    }
    
    return nil
}

// raiseAlarm publishes alarm to NATS and updates Redis
func (lc *LimitChecker) raiseAlarm(ctx context.Context, alarm *Alarm) error {
    // Publish to NATS
    subject := fmt.Sprintf("telemetry.alarms.%s", alarm.SatelliteID)
    data, _ := json.Marshal(alarm)
    lc.jetStream.Publish(subject, data)
    
    // Update Redis active alarms set
    key := fmt.Sprintf("alarms:active:%s", alarm.SatelliteID)
    lc.redis.HSet(ctx, key, alarm.Parameter, string(data))
    
    lc.metrics.AlarmsRaised.Inc()
    
    return nil
}
```

```go
// ArchiveOldData moves data from TimescaleDB to S3
func (dlm *DataLifecycleManager) ArchiveOldData(ctx context.Context, cfg ArchiveConfig) error {
    cutoff := time.Now().Add(-cfg.ArchiveAfter)
    
    // Query data to archive
    query := fmt.Sprintf(`
        SELECT * FROM %s 
        WHERE time < $1 
        ORDER BY satellite_id, time
    `, cfg.SourceTable)
    
    rows, err := dlm.db.Query(ctx, query, cutoff)
    if err != nil {
        return fmt.Errorf("query failed: %w", err)
    }
    defer rows.Close()
    
    // Group by partition (day/month) and satellite
    partitions := make(map[string][]TelemetryRecord)
    
    for rows.Next() {
        var record TelemetryRecord
        if err := rows.Scan(&record); err != nil {
            continue
        }
        
        key := dlm.partitionKey(record, cfg.PartitionBy)
        partitions[key] = append(partitions[key], record)
    }
    
    // Write each partition to S3
    for key, records := range partitions {
        s3Key := fmt.Sprintf("telemetry/%s/%s.parquet", 
            key, uuid.New().String())
        
        // Convert to Parquet
        parquetData, err := dlm.toParquet(records)
        if err != nil {
            dlm.metrics.ArchiveErrors.Inc()
            continue
        }
        
        // Upload to S3
        _, err = dlm.s3Client.PutObject(ctx, &s3.PutObjectInput{
            Bucket:       aws.String(dlm.bucket),
            Key:          aws.String(s3Key),
            Body:         bytes.NewReader(parquetData),
            ContentType:  aws.String("application/vnd.apache.parquet"),
            StorageClass: types.StorageClassIntelligentTiering,
        })
        
        if err != nil {
            dlm.metrics.ArchiveErrors.Inc()
            continue
        }
        
        dlm.metrics.RecordsArchived.Add(float64(len(records)))
    }
    
    // Delete archived data if configured
    if cfg.DeleteAfter {
        _, err = dlm.db.Exec(ctx, 
            fmt.Sprintf("DELETE FROM %s WHERE time < $1", cfg.SourceTable),
            cutoff,
        )
        if err != nil {
            return fmt.Errorf("delete failed: %w", err)
        }
    }
    
    return nil
}

// QueryArchive retrieves data from S3 archive
func (dlm *DataLifecycleManager) QueryArchive(ctx context.Context, 
    satelliteID string, start, end time.Time) ([]TelemetryRecord, error) {
    
    // List relevant S3 objects
    prefix := fmt.Sprintf("telemetry/%s/", satelliteID)
    
    paginator := s3.NewListObjectsV2Paginator(dlm.s3Client, &s3.ListObjectsV2Input{
        Bucket: aws.String(dlm.bucket),
        Prefix: aws.String(prefix),
    })
    
    var records []TelemetryRecord
    
    for paginator.HasMorePages() {
        page, err := paginator.NextPage(ctx)
        if err != nil {
            return nil, err
        }
        
        for _, obj := range page.Contents {
            // Parse date from key and filter
            objDate := dlm.parseDateFromKey(*obj.Key)
            if objDate.Before(start) || objDate.After(end) {
                continue
            }
            
            // Download and parse Parquet file
            result, err := dlm.s3Client.GetObject(ctx, &s3.GetObjectInput{
                Bucket: aws.String(dlm.bucket),
                Key:    obj.Key,
            })
            if err != nil {
                continue
            }
            
            pageRecords, err := dlm.fromParquet(result.Body)
            if err != nil {
                continue
            }
            
            // Filter by exact time range
            for _, r := range pageRecords {
                if r.Time.After(start) && r.Time.Before(end) {
                    records = append(records, r)
                }
            }
        }
    }
    
    return records, nil
}
```

```go
// websocket_gateway.go

package telemetry

import (
    "context"
    "encoding/json"
    "sync"
    "time"
    
    "github.com/gorilla/websocket"
    "github.com/nats-io/nats.go"
    "github.com/redis/go-redis/v9"
)

// Client represents a connected WebSocket client
type Client struct {
    ID            string
    Conn          *websocket.Conn
    Subscriptions map[string]*Subscription
    Send          chan []byte
    mu            sync.RWMutex
    lastPing      time.Time
}

// Subscription represents a telemetry subscription
type Subscription struct {
    ID          string
    SatelliteID string
    Parameters  []string
    Subsystems  []string
    Alarms      bool
    NatsSub     *nats.Subscription
}

// WebSocketGateway manages client connections and message routing
type WebSocketGateway struct {
    clients     map[string]*Client
    mu          sync.RWMutex
    natsConn    *nats.Conn
    redis       *redis.Client
    upgrader    websocket.Upgrader
    metrics     *GatewayMetrics
    maxClients  int
    maxSubsPerClient int
}

// NewWebSocketGateway creates a new gateway instance
func NewWebSocketGateway(cfg GatewayConfig) (*WebSocketGateway, error) {
    nc, err := nats.Connect(cfg.NatsURL)
    if err != nil {
        return nil, fmt.Errorf("NATS connection failed: %w", err)
    }
    
    rdb := redis.NewClient(&redis.Options{
        Addr: cfg.RedisAddr,
    })
    
    return &WebSocketGateway{
        clients:    make(map[string]*Client),
        natsConn:   nc,
        redis:      rdb,
        upgrader: websocket.Upgrader{
            ReadBufferSize:  1024,
            WriteBufferSize: 1024,
            CheckOrigin: func(r *http.Request) bool {
                return true // Configure properly in production
            },
        },
        metrics:         NewGatewayMetrics(),
        maxClients:      cfg.MaxClients,
        maxSubsPerClient: cfg.MaxSubscriptionsPerClient,
    }, nil
}

// HandleConnection upgrades HTTP to WebSocket and manages client
func (gw *WebSocketGateway) HandleConnection(w http.ResponseWriter, r *http.Request) {
    // Check client limit
    gw.mu.RLock()
    if len(gw.clients) >= gw.maxClients {
        gw.mu.RUnlock()
        http.Error(w, "Server at capacity", http.StatusServiceUnavailable)
        return
    }
    gw.mu.RUnlock()
    
    // Upgrade connection
    conn, err := gw.upgrader.Upgrade(w, r, nil)
    if err != nil {
        return
    }
    
    client := &Client{
        ID:            uuid.New().String(),
        Conn:          conn,
        Subscriptions: make(map[string]*Subscription),
        Send:          make(chan []byte, 256),
        lastPing:      time.Now(),
    }
    
    gw.mu.Lock()
    gw.clients[client.ID] = client
    gw.mu.Unlock()
    
    gw.metrics.ClientsConnected.Inc()
    
    // Start client goroutines
    go gw.writePump(client)
    go gw.readPump(client)
}

// readPump handles incoming messages from client
func (gw *WebSocketGateway) readPump(client *Client) {
    defer func() {
        gw.disconnectClient(client)
    }()
    
    client.Conn.SetReadLimit(4096)
    client.Conn.SetReadDeadline(time.Now().Add(60 * time.Second))
    client.Conn.SetPongHandler(func(string) error {
        client.lastPing = time.Now()
        client.Conn.SetReadDeadline(time.Now().Add(60 * time.Second))
        return nil
    })
    
    for {
        _, message, err := client.Conn.ReadMessage()
        if err != nil {
            if websocket.IsUnexpectedCloseError(err, 
                websocket.CloseGoingAway, websocket.CloseAbnormalClosure) {
                gw.metrics.ReadErrors.Inc()
            }
            return
        }
        
        gw.handleMessage(client, message)
    }
}

// writePump sends messages to client
func (gw *WebSocketGateway) writePump(client *Client) {
    ticker := time.NewTicker(30 * time.Second)
    defer func() {
        ticker.Stop()
        client.Conn.Close()
    }()
    
    for {
        select {
        case message, ok := <-client.Send:
            client.Conn.SetWriteDeadline(time.Now().Add(10 * time.Second))
            if !ok {
                client.Conn.WriteMessage(websocket.CloseMessage, []byte{})
                return
            }
            
            w, err := client.Conn.NextWriter(websocket.TextMessage)
            if err != nil {
                return
            }
            w.Write(message)
            
            // Batch queued messages
            n := len(client.Send)
            for i := 0; i < n; i++ {
                w.Write([]byte{'\n'})
                w.Write(<-client.Send)
            }
            
            if err := w.Close(); err != nil {
                return
            }
            
        case <-ticker.C:
            client.Conn.SetWriteDeadline(time.Now().Add(10 * time.Second))
            if err := client.Conn.WriteMessage(websocket.PingMessage, nil); err != nil {
                return
            }
        }
    }
}

// handleMessage processes incoming client message
func (gw *WebSocketGateway) handleMessage(client *Client, data []byte) {
    var msg struct {
        Type    string          `json:"type"`
        ID      string          `json:"id"`
        Payload json.RawMessage `json:"payload"`
    }
    
    if err := json.Unmarshal(data, &msg); err != nil {
        gw.sendError(client, "", "INVALID_MESSAGE", "Failed to parse message")
        return
    }
    
    switch msg.Type {
    case "subscribe":
        gw.handleSubscribe(client, msg.ID, msg.Payload)
    case "unsubscribe":
        gw.handleUnsubscribe(client, msg.ID)
    case "request_current":
        gw.handleRequestCurrent(client, msg.ID, msg.Payload)
    default:
        gw.sendError(client, msg.ID, "UNKNOWN_TYPE", "Unknown message type")
    }
}

// handleSubscribe creates a new subscription
func (gw *WebSocketGateway) handleSubscribe(client *Client, id string, payload json.RawMessage) {
    var req struct {
        SatelliteID string   `json:"satellite_id"`
        Parameters  []string `json:"parameters"`
        Subsystems  []string `json:"subsystems"`
        Alarms      bool     `json:"include_alarms"`
    }
    
    if err := json.Unmarshal(payload, &req); err != nil {
        gw.sendError(client, id, "INVALID_PAYLOAD", "Failed to parse subscription")
        return
    }
    
    // Check subscription limit
    client.mu.RLock()
    if len(client.Subscriptions) >= gw.maxSubsPerClient {
        client.mu.RUnlock()
        gw.sendError(client, id, "LIMIT_EXCEEDED", "Maximum subscriptions reached")
        return
    }
    client.mu.RUnlock()
    
    // Build NATS subject pattern
    var subject string
    if len(req.Parameters) > 0 {
        subject = fmt.Sprintf("telemetry.params.%s.*", req.SatelliteID)
    } else if len(req.Subsystems) > 0 {
        subject = fmt.Sprintf("telemetry.params.%s.%s", 
            req.SatelliteID, req.Subsystems[0]) // TODO: multiple subsystems
    } else {
        subject = fmt.Sprintf("telemetry.params.%s.*", req.SatelliteID)
    }
    
    // Subscribe to NATS
    natsSub, err := gw.natsConn.Subscribe(subject, func(msg *nats.Msg) {
        // Filter by specific parameters if needed
        var param CalibratedParameter
        if err := json.Unmarshal(msg.Data, &param); err != nil {
            return
        }
        
        if len(req.Parameters) > 0 && !contains(req.Parameters, param.Name) {
            return
        }
        
        // Send to client
        wsMsg := map[string]interface{}{
            "type":      "telemetry",
            "timestamp": param.Timestamp,
            "payload":   param,
        }
        data, _ := json.Marshal(wsMsg)
        
        select {
        case client.Send <- data:
        default:
            gw.metrics.DroppedMessages.Inc()
        }
    })
    
    if err != nil {
        gw.sendError(client, id, "SUBSCRIBE_FAILED", err.Error())
        return
    }
    
    // Subscribe to alarms if requested
    var alarmSub *nats.Subscription
    if req.Alarms {
        alarmSubject := fmt.Sprintf("telemetry.alarms.%s", req.SatelliteID)
        alarmSub, _ = gw.natsConn.Subscribe(alarmSubject, func(msg *nats.Msg) {
            wsMsg := map[string]interface{}{
                "type":      "alarm",
                "timestamp": time.Now().UTC(),
                "payload":   json.RawMessage(msg.Data),
            }
            data, _ := json.Marshal(wsMsg)
            
            select {
            case client.Send <- data:
            default:
                gw.metrics.DroppedMessages.Inc()
            }
        })
    }
    
    // Store subscription
    sub := &Subscription{
        ID:          id,
        SatelliteID: req.SatelliteID,
        Parameters:  req.Parameters,
        Subsystems:  req.Subsystems,
        Alarms:      req.Alarms,
        NatsSub:     natsSub,
    }
    
    client.mu.Lock()
    client.Subscriptions[id] = sub
    client.mu.Unlock()
    
    gw.metrics.ActiveSubscriptions.Inc()
    
    // Send confirmation
    response := map[string]interface{}{
        "type": "subscribed",
        "id":   id,
        "payload": map[string]interface{}{
            "satellite_id": req.SatelliteID,
        },
    }
    data, _ := json.Marshal(response)
    client.Send <- data
}

// handleRequestCurrent sends current values snapshot
func (gw *WebSocketGateway) handleRequestCurrent(client *Client, id string, payload json.RawMessage) {
    var req struct {
        SatelliteID string   `json:"satellite_id"`
        Parameters  []string `json:"parameters"`
    }
    
    if err := json.Unmarshal(payload, &req); err != nil {
        gw.sendError(client, id, "INVALID_PAYLOAD", "Failed to parse request")
        return
    }
    
    ctx := context.Background()
    
    // Get current values from Redis
    pattern := fmt.Sprintf("current:%s:*", req.SatelliteID)
    keys, err := gw.redis.Keys(ctx, pattern).Result()
    if err != nil {
        gw.sendError(client, id, "REDIS_ERROR", err.Error())
        return
    }
    
    params := make([]map[string]interface{}, 0, len(keys))
    for _, key := range keys {
        data, err := gw.redis.Get(ctx, key).Result()
        if err != nil {
            continue
        }
        
        var param map[string]interface{}
        if err := json.Unmarshal([]byte(data), &param); err != nil {
            continue
        }
        
        // Filter if specific parameters requested
        if len(req.Parameters) > 0 && req.Parameters[0] != "*" {
            paramName, _ := param["name"].(string)
            if !contains(req.Parameters, paramName) {
                continue
            }
        }
        
        params = append(params, param)
    }
    
    // Send response
    response := map[string]interface{}{
        "type":      "current",
        "id":        id,
        "timestamp": time.Now().UTC(),
        "payload": map[string]interface{}{
            "satellite_id": req.SatelliteID,
            "parameters":   params,
        },
    }
    data, _ := json.Marshal(response)
    client.Send <- data
}
```

### 9.2 Automatic Resolution Selection

```go
// query_optimizer.go

package telemetry

import (
    "time"
)

// QueryOptimizer selects optimal data source and resolution
type QueryOptimizer struct {
    hotRetention  time.Duration  // 7 days
    warmRetention time.Duration  // 90 days
}

// DataSource represents where to query from
type DataSource string

const (
    DataSourceHot  DataSource = "telemetry_raw"
    DataSourceWarm DataSource = "telemetry_1min"
    DataSourceCold DataSource = "telemetry_1hour"
    DataSourceS3   DataSource = "s3_archive"
)

// OptimizeQuery determines the best data source and resolution
func (qo *QueryOptimizer) OptimizeQuery(req *QueryHistoryRequest) (*OptimizedQuery, error) {
    duration := req.EndTime.Sub(req.StartTime)
    age := time.Since(req.EndTime)
    
    var source DataSource
    var resolution string
    var table string
    
    // Select data source based on age
    if age < qo.hotRetention {
        // Recent data - use raw or specified resolution
        if duration < 1*time.Hour {
            source = DataSourceHot
            resolution = "raw"
            table = "telemetry_raw"
        } else if duration < 24*time.Hour {
            source = DataSourceWarm
            resolution = "1min"
            table = "telemetry_1min"
        } else {
            source = DataSourceWarm
            resolution = "1min"
            table = "telemetry_1min"
        }
    } else if age < qo.warmRetention {
        // Medium-term data - use aggregates
        if duration < 7*24*time.Hour {
            source = DataSourceWarm
            resolution = "1min"
            table = "telemetry_1min"
        } else {
            source = DataSourceCold
            resolution = "1hour"
            table = "telemetry_1hour"
        }
    } else {
        // Old data - use S3 archive
        source = DataSourceS3
        resolution = "1hour"
        table = "" // Will query S3
    }
    
    // Override with explicit resolution if provided
    if req.Resolution != Resolution_RAW {
        resolution = resolutionToString(req.Resolution)
        table = resolutionToTable(req.Resolution)
    }
    
    // Estimate result size and warn if large
    estimatedSamples := qo.estimateSamples(duration, resolution, len(req.Parameters))
    
    return &OptimizedQuery{
        Source:           source,
        Resolution:       resolution,
        Table:            table,
        EstimatedSamples: estimatedSamples,
        NeedsS3:          source == DataSourceS3,
    }, nil
}

// estimateSamples estimates the number of samples in result
func (qo *QueryOptimizer) estimateSamples(duration time.Duration, resolution string, paramCount int) int64 {
    var samplesPerParam int64
    
    switch resolution {
    case "raw":
        samplesPerParam = int64(duration.Seconds()) // ~1 Hz assumed
    case "1min":
        samplesPerParam = int64(duration.Minutes())
    case "1hour":
        samplesPerParam = int64(duration.Hours())
    }
    
    if paramCount == 0 {
        paramCount = 100 // Assume 100 params if not specified
    }
    
    return samplesPerParam * int64(paramCount)
}
```

```go
// QueryHistory retrieves historical telemetry data
func (qs *TelemetryQueryService) QueryHistory(
    ctx context.Context, 
    req *QueryHistoryRequest,
) (*QueryHistoryResponse, error) {
    
    startTime := time.Now()
    
    // Optimize query
    optimized, err := qs.optimizer.OptimizeQuery(req)
    if err != nil {
        return nil, fmt.Errorf("query optimization failed: %w", err)
    }
    
    // Warn if large result set
    if optimized.EstimatedSamples > 100000 {
        // Consider suggesting streaming or export
    }
    
    var results []*ParameterHistory
    
    if optimized.NeedsS3 {
        // Query S3 archive
        results, err = qs.queryS3Archive(ctx, req)
    } else {
        // Query TimescaleDB
        results, err = qs.queryTimescaleDB(ctx, req, optimized)
    }
    
    if err != nil {
        return nil, err
    }
    
    queryDuration := time.Since(startTime)
    qs.metrics.QueryDuration.Observe(queryDuration.Seconds())
    
    return &QueryHistoryResponse{
        Parameters: results,
        Metadata: &QueryMetadata{
            QueryTime:   queryDuration,
            DataSource:  string(optimized.Source),
        },
    }, nil
}

// queryTimescaleDB executes query against TimescaleDB
func (qs *TelemetryQueryService) queryTimescaleDB(
    ctx context.Context,
    req *QueryHistoryRequest,
    optimized *OptimizedQuery,
) ([]*ParameterHistory, error) {
    
    // Build query based on table
    var query string
    
    if optimized.Table == "telemetry_raw" {
        query = `
            SELECT 
                time, parameter_name, mnemonic, unit,
                eng_value, limit_status, quality
            FROM telemetry_raw
            WHERE satellite_id = $1
              AND time >= $2 AND time < $3
              AND ($4::text[] IS NULL OR parameter_name = ANY($4))
            ORDER BY parameter_name, time
            LIMIT $5
        `
    } else {
        // Aggregated table
        query = fmt.Sprintf(`
            SELECT 
                bucket as time, parameter_name, mnemonic, unit,
                min_value, max_value, avg_value, first_value, last_value,
                sample_count
            FROM %s
            WHERE satellite_id = $1
              AND bucket >= $2 AND bucket < $3
              AND ($4::text[] IS NULL OR parameter_name = ANY($4))
            ORDER BY parameter_name, bucket
            LIMIT $5
        `, optimized.Table)
    }
    
    // Convert parameters to array
    var params []string
    if len(req.Parameters) > 0 {
        params = req.Parameters
    }
    
    limit := req.Limit
    if limit == 0 {
        limit = 10000 // Default limit
    }
    
    rows, err := qs.db.Query(ctx, query,
        req.SatelliteId,
        req.StartTime,
        req.EndTime,
        params,
        limit,
    )
    if err != nil {
        return nil, fmt.Errorf("query failed: %w", err)
    }
    defer rows.Close()
    
    // Group results by parameter
    resultsMap := make(map[string]*ParameterHistory)
    
    for rows.Next() {
        var (
            timestamp     time.Time
            paramName     string
            mnemonic      string
            unit          string
            value         float64
            limitStatus   string
            quality       []byte
        )
        
        if optimized.Table == "telemetry_raw" {
            err = rows.Scan(&timestamp, &paramName, &mnemonic, &unit,
                &value, &limitStatus, &quality)
        } else {
            var minVal, maxVal, avgVal, firstVal, lastVal float64
            var count int64
            err = rows.Scan(&timestamp, &paramName, &mnemonic, &unit,
                &minVal, &maxVal, &avgVal, &firstVal, &lastVal, &count)
            value = avgVal // Use avg for aggregated
        }
        
        if err != nil {
            continue
        }
        
        // Get or create parameter history
        ph, ok := resultsMap[paramName]
        if !ok {
            ph = &ParameterHistory{
                Name:     paramName,
                Mnemonic: mnemonic,
                Unit:     unit,
                Samples:  make([]*TelemetrySample, 0),
            }
            resultsMap[paramName] = ph
        }
        
        ph.Samples = append(ph.Samples, &TelemetrySample{
            Timestamp:   timestamp,
            Value:       value,
            LimitStatus: limitStatus,
        })
    }
    
    // Convert map to slice
    results := make([]*ParameterHistory, 0, len(resultsMap))
    for _, ph := range resultsMap {
        results = append(results, ph)
    }
    
    return results, nil
}

// GetCurrentValue retrieves the latest value from Redis
func (qs *TelemetryQueryService) GetCurrentValue(
    ctx context.Context,
    req *GetCurrentValueRequest,
) (*GetCurrentValueResponse, error) {
    
    key := fmt.Sprintf("current:%s:%s", req.SatelliteId, req.Parameter)
    
    data, err := qs.redis.Get(ctx, key).Result()
    if err == redis.Nil {
        return nil, fmt.Errorf("parameter not found: %s", req.Parameter)
    }
    if err != nil {
        return nil, fmt.Errorf("redis error: %w", err)
    }
    
    var param CalibratedParameter
    if err := json.Unmarshal([]byte(data), &param); err != nil {
        return nil, fmt.Errorf("unmarshal error: %w", err)
    }
    
    return &GetCurrentValueResponse{
        Parameter: &param,
    }, nil
}
```

### 15.3 Alerting Rules

```yaml
# prometheus-alerts.yaml

groups:
  - name: telemetry-pipeline
    rules:
      - alert: TelemetryProcessingBacklog
        expr: telemetry_processing_queue_depth > 500
        for: 2m
        labels:
          severity: warning
        annotations:
          summary: "Telemetry processing backlog"
          description: "Queue depth {{ $value }} exceeds threshold"
          
      - alert: TelemetryLatencyHigh
        expr: histogram_quantile(0.95, telemetry_processing_duration_seconds) > 0.5
        for: 5m
        labels:
          severity: warning
        annotations:
          summary: "High telemetry latency"
          description: "P95 latency {{ $value }}s exceeds 500ms threshold"
          
      - alert: TelemetryFrameErrors
        expr: rate(telemetry_frames_errors_total[5m]) > 10
        for: 2m
        labels:
          severity: warning
        annotations:
          summary: "High frame error rate"
          description: "{{ $value }} errors/sec on {{ $labels.station_id }}"
          
      - alert: NoTelemetryReceived
        expr: rate(telemetry_frames_received_total[5m]) == 0
        for: 5m
        labels:
          severity: critical
        annotations:
          summary: "No telemetry received"
          description: "Station {{ $labels.station_id }} not receiving frames"
          
      - alert: DatabaseWriteErrors
        expr: rate(telemetry_db_errors_total[5m]) > 1
        for: 2m
        labels:
          severity: critical
        annotations:
          summary: "Database write errors"
          description: "{{ $value }} write errors/sec"
```
