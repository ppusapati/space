## 1. Executive Summary

### 1.1 Purpose

The Real-Time WebSocket Protocol provides bidirectional, low-latency communication between P9E Space Platform servers and clients. It enables:

- **Telemetry Streaming:** Real-time satellite parameter updates
- **Pass Status:** Live pass execution state changes
- **Alerts & Alarms:** Immediate notification of limit violations
- **Command Status:** Real-time command execution feedback
- **System Events:** Platform-wide notifications

### 1.2 Design Goals

| Goal | Target | Rationale |
|------|--------|-----------|
| **Low Latency** | < 100ms server-to-client | Real-time operations |
| **High Throughput** | 100K msgs/sec aggregate | Fleet-scale telemetry |
| **Scalability** | 50K concurrent connections | Multi-tenant platform |
| **Reliability** | 99.9% message delivery | Mission-critical data |
| **Efficiency** | Minimal bandwidth | Mobile/constrained clients |

### 1.3 Key Features

| Feature | Description |
|---------|-------------|
| **Topic-based Subscriptions** | Subscribe to specific data streams |
| **Multiplexed Channels** | Multiple logical channels per connection |
| **Binary & JSON Messages** | Format flexibility |
| **Automatic Reconnection** | Client resilience |
| **Backpressure Handling** | Graceful overload management |
| **Per-message Compression** | Bandwidth optimization |

## 3. Protocol Specification

### 3.1 Connection Endpoint

```
Production:  wss://api.p9e.space/v1/ws
Staging:     wss://api.staging.p9e.space/v1/ws
Development: ws://localhost:8080/v1/ws
```

### 3.2 URL Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `token` | Optional | JWT token (alternative to header) |
| `version` | Optional | Protocol version (default: 1) |
| `compress` | Optional | Enable compression (default: true) |

**Example:**
```
wss://api.p9e.space/v1/ws?token=eyJhbG...&compress=true
```

### 3.3 Protocol Versions

| Version | Status | Features |
|---------|--------|----------|
| 1 | Current | Base protocol, JSON messages |
| 2 | Planned | Binary messages, MessagePack |

### 3.4 Message Frame Format

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         MESSAGE FRAME FORMAT                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  All messages are JSON objects with the following structure:                │
│                                                                             │
│  {                                                                          │
│    "type": "<message_type>",           // Required: Message type           │
│    "id": "<correlation_id>",           // Optional: For request/response   │
│    "channel": "<channel_name>",        // Optional: Target channel         │
│    "timestamp": "<ISO8601>",           // Optional: Server timestamp       │
│    "payload": { ... }                  // Required: Message-specific data  │

│  {                                                                          │
│    "type": "telemetry",                                                    │
```

## 4. Channel Definitions

### 4.1 Channel Hierarchy

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          CHANNEL HIERARCHY                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  telemetry                                                                  │
│  ├── telemetry.{satellite_id}                    // All params for sat     │
│  │   ├── telemetry.{satellite_id}.{subsystem}    // Subsystem params       │
│  │   │   └── telemetry.{satellite_id}.{subsystem}.{param}  // Single param │
│  │   └── telemetry.{satellite_id}.summary       // Key params only         │
│  │                                                                          │
│  alerts                                                                     │
│  ├── alerts.{satellite_id}                       // Alerts for satellite   │
│  ├── alerts.{station_id}                         // Alerts for station     │
│  └── alerts.all                                  // All alerts (operators) │
│                                                                             │
│  passes                                                                     │
│  ├── passes.{pass_id}                            // Single pass updates    │
│  ├── passes.{station_id}                         // Station's passes       │
│  └── passes.active                               // All active passes      │
│                                                                             │
│  commands                                                                   │
│  ├── commands.{satellite_id}                     // Command status         │
│  └── commands.{command_id}                       // Single command         │
│                                                                             │
│  system                                                                     │
│  ├── system.health                               // System health          │
│  ├── system.maintenance                          // Maintenance notices    │
│  └── system.announcements                        // Platform announcements │
│                                                                             │
│  user                                                                       │
│  └── user.{user_id}                              // User-specific notifs   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 4.2 Channel Patterns

| Pattern | Description | Example |
|---------|-------------|---------|
| `*` | Single segment wildcard | `telemetry.SAT-001.*` |
| `>` | Multi-segment wildcard | `alerts.>` |
| `{var}` | Variable substitution | `passes.{pass_id}` |

### 4.3 Channel Access Control

```yaml
# Channel permissions by role

roles:
  operator:
    subscribe:
      - telemetry.>
      - alerts.>
      - passes.>
      - commands.>
      - system.>
    publish:
      - commands.{satellite_id}
      
  analyst:
    subscribe:
      - telemetry.{satellite_id}.summary
      - alerts.{satellite_id}
      - passes.active
      - system.announcements
    publish: []
    
  viewer:
    subscribe:
      - telemetry.{satellite_id}.summary
      - system.announcements
    publish: []
    
  system:
    subscribe:
      - "*"
    publish:
      - "*"
```

### 4.4 Channel Rate Limits

| Channel Pattern | Max Messages/sec | Burst |
|-----------------|------------------|-------|
| `telemetry.{sat}` | 1000 | 5000 |
| `telemetry.{sat}.{ss}` | 100 | 500 |
| `alerts.*` | 100 | 1000 |
| `passes.*` | 50 | 100 |
| `commands.*` | 50 | 100 |
| `system.*` | 10 | 50 |

## 6. Authentication & Authorization

### 6.1 Authentication Methods

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      AUTHENTICATION METHODS                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  METHOD 1: Token in HTTP Header (Recommended)                              │
│  ═══════════════════════════════════════════                               │
│                                                                             │
│  GET /v1/ws HTTP/1.1                                                       │
│  Host: api.p9e.space                                                       │
│  Upgrade: websocket                                                        │
│  Connection: Upgrade                                                       │
│  Authorization: Bearer eyJhbGciOiJSUzI1NiIs...                            │
│                                                                             │
│  ───────────────────────────────────────────────────────────────────────   │
│                                                                             │
│  METHOD 2: Token in URL Parameter                                          │
│  ═════════════════════════════════                                         │
│                                                                             │
│  wss://api.p9e.space/v1/ws?token=eyJhbGciOiJSUzI1NiIs...                  │
│                                                                             │
│  Note: Less secure, token may appear in logs. Use only when headers       │
│        are not available (some WebSocket libraries).                       │
│                                                                             │
│  ───────────────────────────────────────────────────────────────────────   │
│                                                                             │
│  METHOD 3: Token in First Message                                          │
│  ════════════════════════════════                                          │
```

```go
// authorization.go

package websocket

// Authorizer checks if client can access channel
type Authorizer struct {
    rules []AuthRule
}

// AuthRule defines access rule
type AuthRule struct {
    ChannelPattern string
    RequiredRoles  []string
    RequiredPerms  []string
    ResourceCheck  func(claims *Claims, channel string) bool
}

// CanSubscribe checks if claims allow subscription
func (a *Authorizer) CanSubscribe(claims *Claims, channel string) bool {
    for _, rule := range a.rules {
        if matchChannel(rule.ChannelPattern, channel) {
            // Check roles
            if !hasAnyRole(claims.Roles, rule.RequiredRoles) {
                return false
            }
            
            // Check permissions
            if !hasAllPerms(claims.Permissions, rule.RequiredPerms) {
                return false
            }
            
            // Check resource-specific access
            if rule.ResourceCheck != nil {
                if !rule.ResourceCheck(claims, channel) {
                    return false
                }
            }
            
            return true
        }
    }
    
    return false // No matching rule = deny
}
```

```go
// subscription_manager.go

package websocket

import (
    "sync"
    "time"
)

// SubscriptionManager tracks all subscriptions for a connection
type SubscriptionManager struct {
    clientID      string
    subscriptions map[string]*Subscription
    mu            sync.RWMutex
    maxSubs       int
}

// Subscription represents an active subscription
type Subscription struct {
    ID          string
    Channel     string
    Pattern     string              // Compiled pattern for matching
    Options     SubscriptionOptions
    CreatedAt   time.Time
    LastMessage time.Time
    MessageCount int64
    
    // Throttling state
    lastSent    time.Time
    buffer      *RingBuffer
    
    // Batching state
    batchBuffer []Message
    batchTimer  *time.Timer
}

// Subscribe adds a new subscription
func (sm *SubscriptionManager) Subscribe(channel string, opts SubscriptionOptions) (*Subscription, error) {
    sm.mu.Lock()
    defer sm.mu.Unlock()
    
    // Check limit
    if len(sm.subscriptions) >= sm.maxSubs {
        return nil, ErrSubscriptionLimit
    }
    
    // Check for duplicate
    if _, exists := sm.subscriptions[channel]; exists {
        return nil, ErrAlreadySubscribed
    }
    
    sub := &Subscription{
        ID:        generateID(),
        Channel:   channel,
        Pattern:   compilePattern(channel),
        Options:   opts,
        CreatedAt: time.Now(),
        buffer:    NewRingBuffer(opts.BufferSize),
    }
    
    // Setup batching if enabled
    if opts.Batch.Enabled {
        sub.batchBuffer = make([]Message, 0, opts.Batch.MaxSize)
        sub.batchTimer = time.AfterFunc(
            time.Duration(opts.Batch.IntervalMS)*time.Millisecond,
            func() { sm.flushBatch(sub) },
        )
    }
    
    sm.subscriptions[channel] = sub
    return sub, nil
}

// MatchChannel finds subscriptions matching a channel
func (sm *SubscriptionManager) MatchChannel(channel string) []*Subscription {
    sm.mu.RLock()
    defer sm.mu.RUnlock()
    
    matches := make([]*Subscription, 0)
    for _, sub := range sm.subscriptions {
        if matchPattern(sub.Pattern, channel) {
            matches = append(matches, sub)
        }
    }
    return matches
}

// ShouldSend checks throttling and returns true if message should be sent
func (sm *SubscriptionManager) ShouldSend(sub *Subscription, msg Message) bool {
    if sub.Options.ThrottleMS == 0 {
        return true
    }
    
    throttle := time.Duration(sub.Options.ThrottleMS) * time.Millisecond
    if time.Since(sub.lastSent) >= throttle {
        sub.lastSent = time.Now()
        return true
    }
    
    // Buffer the message for later
    if sub.Options.DropOnOverflow {
        sub.buffer.Push(msg)
    }
    
    return false
}
```

### 8.3 Graceful Shutdown

```go
// shutdown.go

package websocket

import (
    "context"
    "time"
)

// GracefulShutdown handles orderly connection termination
func (c *Connection) GracefulShutdown(ctx context.Context, reason string) error {
    c.mu.Lock()
    if c.State == StateClosed || c.State == StateClosing {
        c.mu.Unlock()
        return nil
    }
    c.State = StateClosing
    c.mu.Unlock()
    
    // Send close message with reason
    closeMsg := Message{
        Type: "close",
        Payload: map[string]interface{}{
            "reason": reason,
            "reconnect_allowed": true,
        },
    }
    
    // Try to send close message
    select {
    case c.send <- marshal(closeMsg):
    case <-time.After(1 * time.Second):
        // Timeout sending close message
    }
    
    // Wait for client acknowledgment or timeout
    select {
    case <-ctx.Done():
    case <-time.After(5 * time.Second):
    }
    
    // Close the connection
    return c.Close()
}

// Close terminates the connection
func (c *Connection) Close() error {
    var err error
    c.closeOnce.Do(func() {
        c.mu.Lock()
        c.State = StateClosed
        c.mu.Unlock()
        
        // Close send channel
        close(c.send)
        
        // Close WebSocket
        err = c.conn.Close()
        
        // Cleanup subscriptions
        c.Subscriptions.UnsubscribeAll()
        
        // Emit disconnection event
        emitConnectionClosed(c.Metadata())
    })
    return err
}
```

### 9.2 Heartbeat Implementation

```go
// heartbeat.go

package websocket

import (
    "time"
)

// HeartbeatConfig configures heartbeat behavior
type HeartbeatConfig struct {
    Interval     time.Duration // Time between pings
    Timeout      time.Duration // Max wait for pong
    MaxMissed    int           // Missed pongs before disconnect
    UseProtocol  bool          // Use WebSocket ping vs app-level
}

// DefaultHeartbeatConfig returns default configuration
func DefaultHeartbeatConfig() HeartbeatConfig {
    return HeartbeatConfig{
        Interval:    30 * time.Second,
        Timeout:     10 * time.Second,
        MaxMissed:   2,
        UseProtocol: true,
    }
}

// HeartbeatManager handles connection keep-alive
type HeartbeatManager struct {
    conn        *Connection
    config      HeartbeatConfig
    ticker      *time.Ticker
    missedPongs int
    lastPong    time.Time
    stopChan    chan struct{}
}

// Start begins heartbeat monitoring
func (hm *HeartbeatManager) Start() {
    hm.ticker = time.NewTicker(hm.config.Interval)
    hm.lastPong = time.Now()
    hm.stopChan = make(chan struct{})
    
    go hm.run()
}

func (hm *HeartbeatManager) run() {
    for {
        select {
        case <-hm.stopChan:
            hm.ticker.Stop()
            return
            
        case <-hm.ticker.C:
            hm.sendPing()
        }
    }
}

func (hm *HeartbeatManager) sendPing() {
    if hm.config.UseProtocol {
        // Use WebSocket protocol ping
        hm.conn.conn.SetWriteDeadline(time.Now().Add(hm.config.Timeout))
        if err := hm.conn.conn.WriteMessage(websocket.PingMessage, nil); err != nil {
            hm.handlePingFailure()
            return
        }
    } else {
        // Use application-level ping
        msg := Message{
            Type: "ping",
            ID:   generateID(),
            Timestamp: time.Now().UTC(),
        }
        select {
        case hm.conn.send <- marshal(msg):
        default:
            hm.handlePingFailure()
        }
    }
    
    // Wait for pong
    go hm.waitForPong()
}

func (hm *HeartbeatManager) waitForPong() {
    time.Sleep(hm.config.Timeout)
    
    if time.Since(hm.lastPong) > hm.config.Interval+hm.config.Timeout {
        hm.missedPongs++
        
        if hm.missedPongs >= hm.config.MaxMissed {
            hm.conn.Close()
        }
    }
}

// OnPong handles pong response
func (hm *HeartbeatManager) OnPong() {
    hm.lastPong = time.Now()
    hm.missedPongs = 0
}

// Stop terminates heartbeat monitoring
func (hm *HeartbeatManager) Stop() {
    close(hm.stopChan)
}
```

```go
// backpressure.go

package websocket

import (
    "sync"
    "time"
)

// BackpressureConfig configures flow control
type BackpressureConfig struct {
    // Buffer sizes
    SendBufferSize    int
    ReceiveBufferSize int
    
    // Overflow behavior
    DropOnOverflow    bool
    DropPriority      []string  // Drop these channels first
    
    // Rate limiting
    MaxMessagesPerSec int
    BurstSize         int
    
    // Slow client handling
    SlowClientTimeout time.Duration
    SlowClientAction  string  // "disconnect", "throttle", "drop"
}

// BackpressureManager handles flow control
type BackpressureManager struct {
    conn         *Connection
    config       BackpressureConfig
    sendBuffer   *PriorityBuffer
    rateLimiter  *RateLimiter
    metrics      *BackpressureMetrics
    mu           sync.Mutex
}

// PriorityBuffer manages messages with priorities
type PriorityBuffer struct {
    high   chan Message
    medium chan Message
    low    chan Message
    mu     sync.Mutex
}

// NewPriorityBuffer creates a new priority buffer
func NewPriorityBuffer(size int) *PriorityBuffer {
    return &PriorityBuffer{
        high:   make(chan Message, size/4),
        medium: make(chan Message, size/2),
        low:    make(chan Message, size/4),
    }
}

// Push adds a message with priority
func (pb *PriorityBuffer) Push(msg Message, priority string) bool {
    var ch chan Message
    
    switch priority {
    case "high":
        ch = pb.high
    case "medium":
        ch = pb.medium
    default:
        ch = pb.low
    }
    
    select {
    case ch <- msg:
        return true
    default:
        // Buffer full
        if priority == "high" {
            // For high priority, try to make room
            pb.dropLowest()
            select {
            case ch <- msg:
                return true
            default:
                return false
            }
        }
        return false
    }
}

// Pop retrieves next message by priority
func (pb *PriorityBuffer) Pop() (Message, bool) {
    // Try high priority first
    select {
    case msg := <-pb.high:
        return msg, true
    default:
    }
    
    // Then medium
    select {
    case msg := <-pb.medium:
        return msg, true
    default:
    }
    
    // Finally low
    select {
    case msg := <-pb.low:
        return msg, true
    default:
        return Message{}, false
    }
}

// dropLowest drops a low priority message
func (pb *PriorityBuffer) dropLowest() {
    select {
    case <-pb.low:
    default:
        select {
        case <-pb.medium:
        default:
        }
    }
}

// Send queues a message for sending
func (bm *BackpressureManager) Send(msg Message) error {
    // Check rate limit
    if !bm.rateLimiter.Allow() {
        bm.metrics.RateLimited.Inc()
        if bm.getPriority(msg) != "high" {
            return ErrRateLimited
        }
    }
    
    priority := bm.getPriority(msg)
    
    if !bm.sendBuffer.Push(msg, priority) {
        bm.metrics.Dropped.Inc()
        if priority == "high" {
            return ErrBufferFull
        }
        return nil // Silently drop non-critical
    }
    
    return nil
}

// getPriority determines message priority
func (bm *BackpressureManager) getPriority(msg Message) string {
    switch msg.Type {
    case "alert", "command_status":
        return "high"
    case "pass_update":
        return "medium"
    default:
        return "low"
    }
}

// RateLimiter implements token bucket rate limiting
type RateLimiter struct {
    tokens     float64
    maxTokens  float64
    refillRate float64  // tokens per second
    lastRefill time.Time
    mu         sync.Mutex
}

// Allow checks if request is allowed
func (rl *RateLimiter) Allow() bool {
    rl.mu.Lock()
    defer rl.mu.Unlock()
    
    // Refill tokens
    now := time.Now()
    elapsed := now.Sub(rl.lastRefill).Seconds()
    rl.tokens = min(rl.maxTokens, rl.tokens+elapsed*rl.refillRate)
    rl.lastRefill = now
    
    if rl.tokens >= 1 {
        rl.tokens--
        return true
    }
    
    return false
}
```

```typescript
// Client-side backpressure handling

class WebSocketClient {
  private messageQueue: Message[] = [];
  private processing: boolean = false;
  private maxQueueSize: number = 1000;
  
  onMessage(event: MessageEvent) {
    const msg = JSON.parse(event.data);
    
    // Queue message for processing
    if (this.messageQueue.length >= this.maxQueueSize) {
      // Drop oldest low-priority messages
      this.dropLowPriority();
    }
    
    this.messageQueue.push(msg);
    this.processQueue();
  }
  
  private async processQueue() {
    if (this.processing) return;
    this.processing = true;
    
    while (this.messageQueue.length > 0) {
      const msg = this.messageQueue.shift()!;
      
      // Process with requestAnimationFrame to avoid blocking UI
      await new Promise<void>(resolve => {
        requestAnimationFrame(() => {
          this.handleMessage(msg);
          resolve();
        });
      });
    }
    
    this.processing = false;
  }
  
  private dropLowPriority() {
    // Keep high priority, drop low priority
    this.messageQueue = this.messageQueue.filter(msg => 
      msg.type === 'alert' || msg.type === 'command_status'
    );
  }
}
```

```typescript
// reconnection.ts

interface ReconnectionConfig {
  initialDelay: number;      // 1000ms
  maxDelay: number;          // 30000ms
  multiplier: number;        // 2
  jitter: number;            // 0.25
  maxAttempts: number;       // 10 (0 = infinite)
  resetAfter: number;        // 60000ms (reset attempts after stable)
}

class ReconnectionManager {
  private config: ReconnectionConfig;
  private attempts: number = 0;
  private lastConnect: number = 0;
  private timer: ReturnType<typeof setTimeout> | null = null;
  
  constructor(config: Partial<ReconnectionConfig> = {}) {
    this.config = {
      initialDelay: 1000,
      maxDelay: 30000,
      multiplier: 2,
      jitter: 0.25,
      maxAttempts: 10,
      resetAfter: 60000,
      ...config
    };
  }
  
  onDisconnect(callback: () => Promise<boolean>): void {
    // Check if should reset attempts
    if (Date.now() - this.lastConnect > this.config.resetAfter) {
      this.attempts = 0;
    }
    
    // Check max attempts
    if (this.config.maxAttempts > 0 && 
        this.attempts >= this.config.maxAttempts) {
      this.emit('max_attempts_reached');
      return;
    }
    
    // Calculate delay with exponential backoff
    const baseDelay = Math.min(
      this.config.initialDelay * Math.pow(this.config.multiplier, this.attempts),
      this.config.maxDelay
    );
    
    // Add jitter
    const jitterRange = baseDelay * this.config.jitter;
    const delay = baseDelay + (Math.random() * 2 - 1) * jitterRange;
    
    this.attempts++;
    
    this.emit('reconnecting', {
      attempt: this.attempts,
      delay,
      maxAttempts: this.config.maxAttempts
    });
    
    // Schedule reconnection
    this.timer = setTimeout(async () => {
      const success = await callback();
      
      if (success) {
        this.lastConnect = Date.now();
        this.emit('reconnected');
      } else {
        // Retry
        this.onDisconnect(callback);
      }
    }, delay);
  }
  
  cancel(): void {
    if (this.timer) {
      clearTimeout(this.timer);
      this.timer = null;
    }
  }
  
  reset(): void {
    this.cancel();
    this.attempts = 0;
  }
}

// Usage
const ws = new WebSocketClient();
const reconnect = new ReconnectionManager();

ws.on('close', (event) => {
  if (event.code !== 1000) { // Not normal close
    reconnect.onDisconnect(async () => {
      try {
        await ws.connect();
        return true;
      } catch {
        return false;
      }
    });
  }
});

ws.on('open', () => {
  reconnect.reset();
});
```

```go
// Gateway manages WebSocket connections
type Gateway struct {

}

// GatewayConfig configures the gateway
type GatewayConfig struct {
    MaxConnections        int
    MaxConnectionsPerUser int
    MaxMessageSize        int64
    ReadBufferSize        int
    WriteBufferSize       int
    Compression           bool
    
    Timeouts  TimeoutConfig
    Heartbeat HeartbeatConfig
    Backpressure BackpressureConfig
}

// NewGateway creates a new gateway
func NewGateway(config *GatewayConfig, deps *Dependencies) *Gateway {
    return &Gateway{
        config: config,
        upgrader: websocket.Upgrader{
            ReadBufferSize:  config.ReadBufferSize,
            WriteBufferSize: config.WriteBufferSize,
            CheckOrigin:     deps.OriginChecker,
            EnableCompression: config.Compression,
        },
        connections: make(map[string]*Connection),
        auth:        deps.Authenticator,
        authorizer:  deps.Authorizer,
        router:      NewMessageRouter(),
        pubsub:      deps.PubSubBridge,
        metrics:     NewGatewayMetrics(),
    }
}

// ServeHTTP handles WebSocket upgrade requests
func (g *Gateway) ServeHTTP(w http.ResponseWriter, r *http.Request) {
    // Check connection limit
    g.mu.RLock()
    if len(g.connections) >= g.config.MaxConnections {
        g.mu.RUnlock()
        http.Error(w, "Server at capacity", http.StatusServiceUnavailable)
        return
    }
    g.mu.RUnlock()
    
    // Extract token from header or query
    token := extractToken(r)
    
    // Pre-authenticate if token provided
    var claims *Claims
    if token != "" {
        var err error
        claims, err = g.auth.ValidateToken(token)
        if err != nil {
            http.Error(w, "Invalid token", http.StatusUnauthorized)
            return
        }
        
        // Check per-user connection limit
        if !g.checkUserConnectionLimit(claims.UserID) {
            http.Error(w, "Too many connections", http.StatusTooManyRequests)
            return
        }
    }
    
    // Upgrade connection
    wsConn, err := g.upgrader.Upgrade(w, r, nil)
    if err != nil {
        return
    }
    
    // Create connection wrapper
    conn := NewConnection(wsConn, g.config)
    conn.RemoteAddr = r.RemoteAddr
    conn.UserAgent = r.UserAgent()
    
    if claims != nil {
        conn.UserID = claims.UserID
        conn.TenantID = claims.TenantID
        conn.Roles = claims.Roles
        conn.Authenticated = true
        conn.State = StateAuthenticated
    }
    
    // Register connection
    g.mu.Lock()
    g.connections[conn.ID] = conn
    g.mu.Unlock()
    
    g.metrics.ConnectionsTotal.Inc()
    g.metrics.ConnectionsActive.Inc()
    
    // Start connection handlers
    go g.handleConnection(conn)
}

// handleConnection manages a single connection
func (g *Gateway) handleConnection(conn *Connection) {
    defer func() {
        g.removeConnection(conn)
    }()
    
    // Start heartbeat
    heartbeat := NewHeartbeatManager(conn, g.config.Heartbeat)
    heartbeat.Start()
    defer heartbeat.Stop()
    
    // Start writer goroutine
    go g.writeLoop(conn)
    
    // Read loop
    g.readLoop(conn)
}

// readLoop processes incoming messages
func (g *Gateway) readLoop(conn *Connection) {
    conn.conn.SetReadLimit(g.config.MaxMessageSize)
    conn.conn.SetReadDeadline(time.Now().Add(g.config.Timeouts.Read))
    
    // Handle pong
    conn.conn.SetPongHandler(func(string) error {
        conn.conn.SetReadDeadline(time.Now().Add(g.config.Timeouts.Read))
        return nil
    })
    
    for {
        _, data, err := conn.conn.ReadMessage()
        if err != nil {
            if websocket.IsUnexpectedCloseError(err,
                websocket.CloseGoingAway,
                websocket.CloseAbnormalClosure) {
                g.metrics.ReadErrors.Inc()
            }
            return
        }
        
        conn.MessagesReceived++
        conn.BytesReceived += int64(len(data))
        conn.LastActivity = time.Now()
        
        // Parse and route message
        msg, err := parseMessage(data)
        if err != nil {
            g.handleError(conn, ErrInvalidMessage, "")
            continue
        }
        
        g.router.Route(conn, msg)
    }
}

// writeLoop sends messages to client
func (g *Gateway) writeLoop(conn *Connection) {
    ticker := time.NewTicker(g.config.Heartbeat.Interval)
    defer ticker.Stop()
    
    for {
        select {
        case msg, ok := <-conn.send:
            if !ok {
                conn.conn.WriteMessage(websocket.CloseMessage, nil)
                return
            }
            
            conn.conn.SetWriteDeadline(time.Now().Add(g.config.Timeouts.Write))
            
            if err := conn.conn.WriteMessage(websocket.TextMessage, msg); err != nil {
                g.metrics.WriteErrors.Inc()
                return
            }
            
            conn.MessagesSent++
            conn.BytesSent += int64(len(msg))
            
        case <-ticker.C:
            conn.conn.SetWriteDeadline(time.Now().Add(g.config.Timeouts.Write))
            if err := conn.conn.WriteMessage(websocket.PingMessage, nil); err != nil {
                return
            }
        }
    }
}
```

```go
// router.go

package websocket

// MessageRouter routes messages to handlers
type MessageRouter struct {
    handlers map[string]MessageHandler
}

// MessageHandler processes a message type
type MessageHandler interface {
    Handle(conn *Connection, msg Message) error
}

// NewMessageRouter creates a new router
func NewMessageRouter() *MessageRouter {
    r := &MessageRouter{
        handlers: make(map[string]MessageHandler),
    }
    
    // Register handlers
    r.handlers["auth"] = &AuthHandler{}
    r.handlers["subscribe"] = &SubscribeHandler{}
    r.handlers["unsubscribe"] = &UnsubscribeHandler{}
    r.handlers["ping"] = &PingHandler{}
    r.handlers["request"] = &RequestHandler{}
    
    return r
}

// Route dispatches message to appropriate handler
func (r *MessageRouter) Route(conn *Connection, msg Message) {
    handler, ok := r.handlers[msg.Type]
    if !ok {
        sendError(conn, msg.ID, ErrUnknownType)
        return
    }
    
    // Check authentication for protected message types
    if requiresAuth(msg.Type) && !conn.Authenticated {
        sendError(conn, msg.ID, ErrAuthRequired)
        return
    }
    
    if err := handler.Handle(conn, msg); err != nil {
        sendError(conn, msg.ID, err)
    }
}

// SubscribeHandler handles subscription requests
type SubscribeHandler struct {
    gateway *Gateway
}

func (h *SubscribeHandler) Handle(conn *Connection, msg Message) error {
    var req SubscribeRequest
    if err := mapstructure.Decode(msg.Payload, &req); err != nil {
        return ErrInvalidMessage
    }
    
    results := make([]ChannelResult, 0, len(req.Channels))
    
    for _, channel := range req.Channels {
        // Check authorization
        if !h.gateway.authorizer.CanSubscribe(conn.Claims(), channel) {
            results = append(results, ChannelResult{
                Channel: channel,
                Status:  "denied",
                Error:   "Not authorized",
            })
            continue
        }
        
        // Create subscription
        sub, err := conn.Subscriptions.Subscribe(channel, req.Options)
        if err != nil {
            results = append(results, ChannelResult{
                Channel: channel,
                Status:  "error",
                Error:   err.Error(),
            })
            continue
        }
        
        // Register with pub/sub bridge
        h.gateway.pubsub.Subscribe(conn, channel)
        
        results = append(results, ChannelResult{
            Channel: channel,
            Status:  "active",
        })
    }
    
    // Get current values if requested
    var currentValues map[string]interface{}
    if req.Options.IncludeCurrent {
        currentValues = h.gateway.getCurrentValues(conn, req.Channels)
    }
    
    // Send response
    response := Message{
        Type:      "subscribed",
        ID:        msg.ID,
        Timestamp: time.Now().UTC(),
        Payload: SubscribedResponse{
            Channels:      results,
            CurrentValues: currentValues,
        },
    }
    
    return conn.Send(response)
}
```

```go
// pubsub_bridge.go

package websocket

import (
    "context"
    "sync"
    
    "github.com/redis/go-redis/v9"
    "github.com/nats-io/nats.go"
)

// PubSubBridge connects WebSocket to message backends
type PubSubBridge struct {
    redis       *redis.Client
    nats        *nats.Conn
    gateway     *Gateway
    
    // Channel to connections mapping
    subscribers map[string]map[string]*Connection  // channel -> connID -> conn
    mu          sync.RWMutex
}

// Subscribe registers a connection for a channel
func (b *PubSubBridge) Subscribe(conn *Connection, channel string) {
    b.mu.Lock()
    defer b.mu.Unlock()
    
    if _, exists := b.subscribers[channel]; !exists {
        b.subscribers[channel] = make(map[string]*Connection)
        
        // First subscriber - create backend subscription
        go b.subscribeBackend(channel)
    }
    
    b.subscribers[channel][conn.ID] = conn
}

// Unsubscribe removes a connection from a channel
func (b *PubSubBridge) Unsubscribe(conn *Connection, channel string) {
    b.mu.Lock()
    defer b.mu.Unlock()
    
    if subs, exists := b.subscribers[channel]; exists {
        delete(subs, conn.ID)
        
        // Last subscriber - remove backend subscription
        if len(subs) == 0 {
            delete(b.subscribers, channel)
            go b.unsubscribeBackend(channel)
        }
    }
}

// subscribeBackend creates NATS/Redis subscription
func (b *PubSubBridge) subscribeBackend(channel string) {
    // Map WebSocket channel to NATS subject
    subject := mapToNATSSubject(channel)
    
    sub, err := b.nats.Subscribe(subject, func(msg *nats.Msg) {
        b.fanOut(channel, msg.Data)
    })
    
    if err != nil {
        log.Error("Failed to subscribe to NATS", "channel", channel, "error", err)
        return
    }
    
    // Store subscription for cleanup
    // ...
}

// fanOut distributes message to all subscribers
func (b *PubSubBridge) fanOut(channel string, data []byte) {
    b.mu.RLock()
    subs := b.subscribers[channel]
    b.mu.RUnlock()
    
    if len(subs) == 0 {
        return
    }
    
    // Parse message
    msg, err := parseBackendMessage(data)
    if err != nil {
        return
    }
    
    // Create WebSocket message
    wsMsg := Message{
        Type:      msg.Type,
        Channel:   channel,
        Timestamp: msg.Timestamp,
        Payload:   msg.Payload,
    }
    
    wsData := marshal(wsMsg)
    
    // Send to all subscribers
    for _, conn := range subs {
        // Check subscription options (throttling, filtering)
        sub := conn.Subscriptions.Get(channel)
        if sub != nil && conn.Subscriptions.ShouldSend(sub, wsMsg) {
            select {
            case conn.send <- wsData:
                conn.MessagesSent++
            default:
                // Buffer full - handle backpressure
                b.gateway.metrics.DroppedMessages.Inc()
            }
        }
    }
}

// Publish sends a message to a channel (for server-initiated messages)
func (b *PubSubBridge) Publish(channel string, msg Message) error {
    data := marshal(msg)
    
    // Publish to Redis for cross-node distribution
    return b.redis.Publish(context.Background(), 
        "ws:"+channel, data).Err()
}
```

```typescript
// websocket-client.ts

interface WebSocketClientConfig {
  url: string;
  token?: string;
  autoReconnect?: boolean;
  reconnectConfig?: ReconnectionConfig;
  heartbeatInterval?: number;
  debug?: boolean;
}

interface Subscription {
  channel: string;
  options?: SubscriptionOptions;
  callback: (message: any) => void;
}

class P9EWebSocketClient extends EventEmitter {
  private config: WebSocketClientConfig;
  private ws: WebSocket | null = null;
  private state: ConnectionState = 'disconnected';
  private subscriptions: Map<string, Subscription> = new Map();
  private pendingRequests: Map<string, PendingRequest> = new Map();
  private reconnectionManager: ReconnectionManager;
  private heartbeatTimer: ReturnType<typeof setInterval> | null = null;
  private messageQueue: Message[] = [];
  
  constructor(config: WebSocketClientConfig) {
    super();
    this.config = {
      autoReconnect: true,
      heartbeatInterval: 30000,
      debug: false,
      ...config
    };
    this.reconnectionManager = new ReconnectionManager(config.reconnectConfig);
  }
  
  async connect(): Promise<void> {
    if (this.state === 'connected' || this.state === 'connecting') {
      return;
    }
    
    this.state = 'connecting';
    this.emit('connecting');
    
    return new Promise((resolve, reject) => {
      const url = this.buildUrl();
      this.ws = new WebSocket(url);
      
      this.ws.onopen = () => {
        this.state = 'connected';
        this.emit('connected');
        this.startHeartbeat();
        this.flushMessageQueue();
        resolve();
      };
      
      this.ws.onclose = (event) => {
        this.handleClose(event);
      };
      
      this.ws.onerror = (event) => {
        this.emit('error', event);
        reject(new Error('WebSocket connection failed'));
      };
      
      this.ws.onmessage = (event) => {
        this.handleMessage(event);
      };
    });
  }
  
  private buildUrl(): string {
    let url = this.config.url;
    if (this.config.token) {
      url += `?token=${encodeURIComponent(this.config.token)}`;
    }
    return url;
  }
  
  async subscribe(
    channel: string,
    callback: (message: any) => void,
    options?: SubscriptionOptions
  ): Promise<void> {
    const subscription: Subscription = { channel, callback, options };
    this.subscriptions.set(channel, subscription);
    
    if (this.state !== 'connected') {
      return; // Will subscribe on reconnect
    }
    
    const requestId = this.generateId();
    
    return new Promise((resolve, reject) => {
      this.pendingRequests.set(requestId, {
        resolve,
        reject,
        timeout: setTimeout(() => {
          this.pendingRequests.delete(requestId);
          reject(new Error('Subscription timeout'));
        }, 30000)
      });
      
      this.send({
        type: 'subscribe',
        id: requestId,
        payload: {
          channels: [channel],
          options
        }
      });
    });
  }
  
  async unsubscribe(channel: string): Promise<void> {
    this.subscriptions.delete(channel);
    
    if (this.state !== 'connected') {
      return;
    }
    
    const requestId = this.generateId();
    
    this.send({
      type: 'unsubscribe',
      id: requestId,
      payload: {
        channels: [channel]
      }
    });
  }
  
  private handleMessage(event: MessageEvent): void {
    try {
      const msg = JSON.parse(event.data);
      
      if (this.config.debug) {
        console.log('WS received:', msg);
      }
      
      // Handle response to pending request
      if (msg.id && this.pendingRequests.has(msg.id)) {
        const pending = this.pendingRequests.get(msg.id)!;
        clearTimeout(pending.timeout);
        this.pendingRequests.delete(msg.id);
        
        if (msg.type === 'error') {
          pending.reject(new Error(msg.payload.message));
        } else {
          pending.resolve(msg.payload);
        }
        return;
      }
      
      // Handle channel messages
      if (msg.channel) {
        const sub = this.findMatchingSubscription(msg.channel);
        if (sub) {
          sub.callback(msg.payload);
        }
      }
      
      // Emit typed events
      this.emit(msg.type, msg);
      
      // Handle specific message types
      switch (msg.type) {
        case 'pong':
          this.lastPong = Date.now();
          break;
          
        case 'token_expiring':
          this.emit('token_expiring', msg.payload);
          break;
          
        case 'close':
          this.handleServerClose(msg.payload);
          break;
      }
      
    } catch (error) {
      this.emit('error', error);
    }
  }
  
  private handleClose(event: CloseEvent): void {
    this.stopHeartbeat();
    this.state = 'disconnected';
    this.emit('disconnected', event);
    
    if (this.config.autoReconnect && event.code !== 1000) {
      this.reconnectionManager.onDisconnect(async () => {
        try {
          await this.connect();
          await this.resubscribeAll();
          return true;
        } catch {
          return false;
        }
      });
    }
  }
  
  private async resubscribeAll(): Promise<void> {
    for (const [channel, sub] of this.subscriptions) {
      try {
        await this.subscribe(channel, sub.callback, sub.options);
      } catch (error) {
        this.emit('error', { 
          type: 'resubscribe_failed', 
          channel, 
          error 
        });
      }
    }
  }
  
  private send(msg: Message): void {
    if (this.state !== 'connected') {
      this.messageQueue.push(msg);
      return;
    }
    
    if (this.config.debug) {
      console.log('WS send:', msg);
    }
    
    this.ws?.send(JSON.stringify(msg));
  }
  
  private flushMessageQueue(): void {
    while (this.messageQueue.length > 0) {
      const msg = this.messageQueue.shift()!;
      this.send(msg);
    }
  }
  
  private startHeartbeat(): void {
    this.heartbeatTimer = setInterval(() => {
      this.send({
        type: 'ping',
        id: this.generateId(),
        timestamp: new Date().toISOString()
      });
    }, this.config.heartbeatInterval);
  }
  
  private stopHeartbeat(): void {
    if (this.heartbeatTimer) {
      clearInterval(this.heartbeatTimer);
      this.heartbeatTimer = null;
    }
  }
  
  disconnect(): void {
    this.config.autoReconnect = false;
    this.reconnectionManager.cancel();
    this.ws?.close(1000, 'Client disconnect');
  }
  
  private generateId(): string {
    return `${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;
  }
}

// Usage example
const client = new P9EWebSocketClient({
  url: 'wss://api.p9e.space/v1/ws',
  token: 'eyJhbGci...',
  autoReconnect: true,
  debug: true
});

client.on('connected', () => {
  console.log('Connected to P9E WebSocket');
});

client.on('telemetry', (msg) => {
  console.log('Telemetry:', msg.payload);
});

client.on('alert', (msg) => {
  console.log('Alert:', msg.payload);
});

await client.connect();

await client.subscribe('telemetry.SAT-001.EPS', (data) => {
  updateTelemetryDisplay(data.parameters);
}, { 
  include_current: true,
  throttle_ms: 1000 
});
```

### 16.3 Alerting Rules

```yaml
# prometheus-alerts-websocket.yaml

groups:
  - name: websocket-gateway
    rules:
      - alert: WebSocketHighConnectionCount
        expr: sum(ws_connections_active) > 20000
        for: 5m
        labels:
          severity: warning
        annotations:
          summary: "High WebSocket connection count"
          description: "{{ $value }} connections active (threshold: 20000)"
          
      - alert: WebSocketGatewayOverloaded
        expr: ws_connections_active / ws_connections_max > 0.9
        for: 2m
        labels:
          severity: critical
        annotations:
          summary: "WebSocket gateway near capacity"
          description: "Gateway {{ $labels.gateway }} at {{ $value | humanizePercentage }} capacity"
          
      - alert: WebSocketHighDropRate
        expr: rate(ws_messages_dropped_total[5m]) > 100
        for: 2m
        labels:
          severity: warning
        annotations:
          summary: "High message drop rate"
          description: "{{ $value }} messages/sec being dropped"
          
      - alert: WebSocketHighLatency
        expr: histogram_quantile(0.99, ws_message_latency_seconds) > 0.5
        for: 5m
        labels:
          severity: warning
        annotations:
          summary: "High WebSocket message latency"
          description: "P99 latency is {{ $value }}s"
          
      - alert: WebSocketAuthFailures
        expr: rate(ws_auth_failures_total[5m]) > 10
        for: 2m
        labels:
          severity: warning
        annotations:
          summary: "High auth failure rate"
          description: "{{ $value }} auth failures/sec"
```
