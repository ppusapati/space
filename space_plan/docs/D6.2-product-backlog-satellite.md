### US-SAT-001: Configure Spacecraft Bus

**Epic:** E-SAT-001  
**Use Case:** UC-SAT-001

**Story Points:** 5

**As a** spacecraft engineer  
**I want to** configure the spacecraft bus definition  
**So that** the system understands the satellite's architecture

**Acceptance Criteria:**

```gherkin
Scenario: Create spacecraft configuration
  Given a satellite is registered (from GS module)
  When I configure the spacecraft bus:
    | Field            | Value                    |
    | bus_type         | 6U CubeSat               |
    | mass_kg          | 12.5                     |
    | launch_date      | 2024-03-15               |
    | design_life_years| 3                        |
    | manufacturer     | P9E Microsystems         |
  Then the spacecraft configuration is saved
  And I can define subsystems

Scenario: Link to satellite record
  Given a spacecraft configuration exists
  Then it is linked to the satellite record from GS module
  And telemetry/commands reference this configuration

Scenario: Configuration versioning
  Given I modify spacecraft configuration
  When I save changes

```

**Technical Notes:**
- Extends satellite record from Ground Station module
- Support bus types: CubeSat (1U-12U), SmallSat, MicroSat
- Configuration versioned for change tracking
- Link to COSMOS/XTCE database format (future)
